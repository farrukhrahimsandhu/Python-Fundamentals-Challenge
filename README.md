# 🐍 50 Python Programs Challenge

Welcome to my Python Fundamentals repository! I completed this 50-program challenge to build a rock-solid technical foundation in Python, progressing from basic syntax to full Command-Line Interface (CLI) applications with data persistence. 

## 🎯 Objective
To master core Python programming concepts, logic building, and problem-solving as a stepping stone toward Machine Learning and Data Science.

## 🛠️ Concepts & Technologies Covered
* **Basic Logic & Math:** Factorials, Fibonacci series, number conversions (Binary/Octal/Hex), and Armstrong numbers.
* **Data Structures:** Advanced list manipulation, dictionary sorting, and set operations.
* **File Handling (I/O):** Reading, writing, appending, and merging `.txt` and `.csv` files.
* **Data Persistence:** Utilizing `json` modules to store and retrieve user data safely.
* **Modular Programming:** Structuring code using helper functions, CRUD operations, and `__main__` execution blocks.

## 🚀 Key Projects Included
Among the 50 programs, I built several mini-projects to test my real-world logic:
1.  **ATM Simulator:** Multi-account system with PIN verification, transaction limits, and history tracking stored in `accounts.json`.
2.  **Contact Book:** A CRUD application to add, view, update, and delete contacts dynamically.
3.  **Mini Marksheet Generator:** Takes student inputs, calculates grades/percentages, and formats a clean CLI report.
4.  **CLI To-Do List:** Task manager with file persistence.
5.  **Security Tools:** A regex-based password strength checker and random password generator.
